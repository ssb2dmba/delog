## Build

Project build well with Android Studio Electric Eel.

You will need an Android NDK and set in the Android Studio env i.e.: 
```
~$ export ANDROID_NDK_HOME=~/Android/Sdk/ndk/25.1.8937393/
~$ ./android-studio/bin/studio.sh
``
